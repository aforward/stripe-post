# StripePost

A minimal library for posting messages to the Stripe API.

## Installation

Add `stripe_post` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [{:stripe_post, "~> 0.1.0"}]
end
```

Documentation can
be found at [https://hexdocs.pm/stripe_post](https://hexdocs.pm/stripe_post).
